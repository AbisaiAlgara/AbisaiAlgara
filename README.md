## Featured Project

### Framework Estructural Showcase

Documentary showcase of a Python/OpenSeesPy framework for 3D structural modeling, modal analysis, response spectrum analysis, drift checks, demand/capacity evaluation, LRFD design, constructability review and multi-objective optimization.

Main tools:

Python · OpenSeesPy · DEAP · NSGA-II · ETABS · SAP2000 · Streamlit · Excel · Markdown · CSV · JSON

Main applied case:

* Four-story steel building
* Moment frames + Chevron bracing
* Approximate steel weight: 44.84 t
* Maximum global D/C ratio: 0.793
* Maximum service drift: 0.000816
* Roof displacement: 8.35 mm
